# DeepFlow Web 🌊

**DeepFlow Web** 是一个为 ADHD 用户设计的单页面时间管理工具，集成了思维缓冲区、任务管理和灵感记录功能。

## ✨ 特性

- **🧠 思维缓冲区**：快速卸载大脑中的杂乱思绪。
- **⚡ 任务管理**：
  - 专注当下：只展示进行中的任务。
  - 自动折叠：已完成的任务自动沉底并折叠。
  - 子任务拆解：支持无限级子任务。
- **💎 丰饶之海**：捕捉每一个稍纵即逝的灵感。
- **🔒 数据安全**：
  - 数据存储在本地浏览器 (LocalStorage)。
  - 支持一键导出/导入数据备份。
- **🎨 极简设计**：
  - 米白色护眼背景。
  - 垂直流式布局，减少注意力分散。
